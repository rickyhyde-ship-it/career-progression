Setup instructions (click to expand)
  1. Open your Google Sheet → Extensions → Apps Script
  2. Delete any existing code and paste the script below
  3. Click Deploy → New deployment
  4. Type: Web app · Execute as: Me · Who has access: Anyone
  5. Click Deploy, authorise when prompted
  6. Copy the Web app URL and paste it below
function doGet() {
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  var data  = sheet.getDataRange().getValues();
  var headers = data[0];
  var rows = data.slice(1).map(function(row) {
    var obj = {};
    headers.forEach(function(h, i) { obj[h] = row[i]; });
    return obj;
  });
  return ContentService
    .createTextOutput(JSON.stringify(rows))
    .setMimeType(ContentService.MimeType.JSON);
}

⚠ "Anyone" can call this URL — treat it like a password and don't share it publicly. Your Google Sheet itself remains private.
